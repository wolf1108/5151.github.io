# 吃什麼？轉盤抽籤 (Food Picker Wheel)

網址: https://wolf1108.github.io/5151.github.io/eat.html

## 設計原因

這是一個為「選擇困難症」而生的小工具。  
某天中午肚子很餓，卻一直想不到要吃什麼，在各種選項之間猶豫了很久，最後乾脆花了半小時寫了一個轉盤，讓它幫我做決定。

目前轉盤內容是以 **雲林虎尾** 我自己常吃、覺得不錯的店家為主，之後也可以依生活圈改變，調整成可自行輸入選項的版本。  
目標很單純：**把每天最浪費時間的選擇，交給轉盤處理，快點去吃飯。**


## 功能介紹 (Features)

1.  **餐別選擇**：
    *   提供 **午餐 (Lunch)** 與 **晚餐 (Dinner)** 兩種模式。
    *   切換模式時，轉盤上的選項會自動更新，針對不同時段提供合適的餐廳或食物類別。

2.  **轉盤抽籤**：
    *   點擊 **「🎲 開始轉盤」** 按鈕，轉盤會開始旋轉。
    *   具有平滑的減速動畫效果，增加儀式感。

3.  **結果顯示**：
    *   轉盤停止後，畫面下方會自動顯示抽中的結果 (例如：🎉 今天吃：**麥當勞**)。

4.  **內建美食清單**：
    *   收錄了多種常見選擇，包括：大吉祥、必勝客、麥當勞、7-11、全家、以及各類小吃 (滷味、丼飯、蚵仔煎等)。

## 如何使用 (How to Use)

1.  **開啟檔案**：
    *   直接雙擊打開 `吃啥.html` 檔案 (會使用你的預設瀏覽器開啟)。
    
2.  **操作步驟**：
    *   在上方選單選擇 **午餐** 或 **晚餐**。
    *   按下綠色的 **開始轉盤** 按鈕。
    *   等待轉盤停下，查看今天的命定美食！

## 技術說明 (Tech Stack)

*   **HTML5**
*   **CSS3** (使用 Canvas 繪製轉盤，包含陰影與指針樣式)
*   **JavaScript** (處理轉盤邏輯、動畫渲染與隨機抽籤)
